About

Here is a collection of different presentations that I have made. The source for all of these can be found on GitHub.

When on a slidehow press the 'P' key to open the presenation mode. This allows you to see the presentation notes for each slide. To clone the slideshow view (for presenting the slides on another monitor) press the 'C' key.

Credits

These pages are powered by remark.js which is a way to turn markdown documents into slide show presentations. Grunt.js is used to do all of the heavy lifting. Each child page for the presentations have a comment section which is powered by Disqus. Everything is hosted on GitHub Pages.
